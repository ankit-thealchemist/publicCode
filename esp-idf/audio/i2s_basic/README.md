This example is to demostrate the use for the testing of the speaker using the max 98357A I2S ic.
To setup use the connection

| Max 98357A Pin | Esp32 Pins      |
| :------------: | --------------- |
|      LAC      | `GPIO_NUM_15` |
|      BCLK      | `GPIO_NUM_13` |
|      DIN      | `GPIO_NUM_21` |
|      VIN      | 5V              |
|      GND      | GND             |
